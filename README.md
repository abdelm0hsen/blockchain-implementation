# Blockchain Implementation

A beginner blockchain simulation in Python covering blocks, SHA-256 hashing, proof-of-work mining, and chain validation.

## Features

- Block and blockchain classes
- Pending transactions pool
- Proof-of-work mining
- Chain integrity validation

## Project Structure

```
blockchain-implementation/
  main.py
  requirements.txt
  README.md
```

## Run

```bash
python main.py
```

Use the CLI menu to add transactions, mine blocks, and validate the chain.

## Learning Outcomes

- Blockchain core concepts
- Hash linking and immutability
- PoW basics in practice
